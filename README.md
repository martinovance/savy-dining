# Savy Dining

A modern, high-end restaurant web application built with **React**, **TypeScript**, and **Vite**.

## Features
- **Table Reservations**: Intuitive guest booking flow.
- **Digital Menu**: Dynamic, high-fidelity menu display.
- **Secure Payments**: Stripe-ready architecture for reservation holds.
- **Responsive Design**: Tailored for luxury mobile and desktop experiences.

## Tech Stack
- **Framework**: Vite + React 18
- **Language**: TypeScript
- **Styling**: TailwindCSS
- **Icons**: Lucide React

## Getting Started

1. **Clone & Install**:
   ```bash
   git clone https://github.com/martinovance/savy-dining.git
   cd savy-dining
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## Architecture
- `src/components`: UI modules for Menu, Reservations, and Payments.
- `src/App.tsx`: Main view orchestration and navigation logic.
